---
title: "install"
group: "CLI"
---

The `install` command creates the `.chiller` directory used to do building and running the app locally, it pulls the chiller app source from github and installs it's dependencies.

## Options

`-f, --force`: When passed, will forcefuly reinstall chiller even if a `.chiller` directory already exists and its dependencies are already installed.

## Example

```
chiller install
chiller install --force
```
